WEBSITE REQUEST FOR
QUOTATION!

To make request for quotation on ecommerce website, without displaying product price. This is good especially for Enterprise and B2B products sales

Feature Overview

This module will perform to be able to request quotation by website visitors and receive website visitors’ information and quotation request by website owner.

Features Details

The Admin user can define quotation products in product form.

This module helps to hide product price and Add to Cart button of quotation products on Shop.

Users can print out quotation slip immediately after requesting quotation process.

Website Visitors Info

Website users can request for quotation products in website shop.

Website users can create their own user account while requesting product quotation.

Website users just need to fill their information (name,company, project,email,phone number, address) after RFQ Check Out.

Hide Related Price and Add to Cart button

Hiding price and add to cart button in recently viewed product list, compare, wishlist view and product search bar list view

Support by all browsers.

Work with any devices.
